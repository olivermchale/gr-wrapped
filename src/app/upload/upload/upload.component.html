How to use

  1. Export your Goodreads Library
    • Follow this link to take you to the export page
    • Click the 'Export Library' button
    • Wait for the library export to generate
    • Click on the generated export to download it
  2. Import your Goodreads Library using the import button below!
    • Clicking the upload button will allow you to select the file you exported from Goodreads
    • You can optionally provide your public Profile URL for increased data accuracy!
